# my-english

## 2025.02.12

### involvement 

- `n`：**介入、参与**

- `involvement in/with sth`：**参与... 介入...**


### substantial 
- `adj`：**大量的 重大的 实质性的**
- `a substantial amount of`：**大量的**

### literature 
- `n`：**文学(作品) 文献 资料**
- `literature on sth`：**(某一学科)的文献**

### spring 
- `n`：**春天 弹簧 发条**
- `spring from sth`：**源自... 由..造成**
- `spring up`：**迅速出现**

### deduce
- `v`：**推断 推理 演绎**
- `deduce from sth`：**从...推断**
